# Course Portal

A simple web application built using **Node.js**, **Express**, and **EJS** to display technical courses and their details.

## Features

- Home page showing total courses
- Courses page listing all available courses
- Course details page using route parameters
- About page with portal information
- Dynamic routing using Express
- EJS templating and partials (navbar & footer)

## Technologies Used

- Node.js
- Express.js
- EJS (Embedded JavaScript)
- HTML
